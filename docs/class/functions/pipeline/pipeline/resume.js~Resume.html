    Resume

    Handle retrieving the current execution status of the state machine, handling the Flood.IO flood.

    If a terminal status is retrieved, the Resume#perform function will resolve to be false, indicating that the action has completed (either successfully or otherwise).

    public async perform(): Array[Boolean, String] source

    Determine the current status of the Step Functions Flood.IO execution.

    If a terminal status is retrieved, then the function will either return false as well as a summary execution message, or throw an error if the state machine exited on an error.

    If the status indicates the state machine is still executing, then this function will resolve to true.

    Return:

    Array[Boolean, String]

    false and a summary message if execution has finished successfully, true if execution is ongoing.
