    Parameters

    Enables the defining of parameters which are passed as overrides to the flood files which are sent to Flood.IO.

    Parameters can be defined as a static string or as an object containing the "Fn::GetParam" key, which behaves in a similar fashion to the CloudFormation parameter overrides function.
